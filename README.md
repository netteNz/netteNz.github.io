# nettenzOS Portfolio

A macOS-inspired portfolio website with a fully interactive terminal emulator, audio visualization, and project showcase — deployed on GitHub Pages.

## Features

- **macOS Desktop UI** — Draggable, resizable windows with minimize/maximize/close controls and a dock
- **Terminal Emulator** — Interactive shell with command history, tab-completion, and project shortcuts
- **Audio Visualization** — Web Audio API with real-time frequency spectrum and LUFS metering
- **Project Showcase** — Browse projects via the terminal or GUI with live demo links
- **PyScript Integration** — In-browser Python execution via PyScript
- **Google Analytics** — Traffic tracking via gtag.js

## Tech Stack

- Vanilla HTML/CSS/JS, TailwindCSS
- GSAP animations
- Web Audio API
- PyScript (2025.3.1)
- Prism.js for syntax highlighting
- GitHub Pages deployment

## Quick Start

```bash
git clone https://github.com/netteNz/netteNz.github.io.git
cd netteNz.github.io
python -m http.server 8000
# Visit: http://localhost:8000
```

## Terminal Commands

| Command | Description |
|---|---|
| `help` | Show available commands |
| `ls` | List files or projects |
| `cd <dir>` | Change directory |
| `pwd` | Print working directory |
| `cat <file>` | Read a file |
| `whoami` | Display user info |
| `readme` | Show this README |
| `projects` | Open projects window |
| `neofetch` | Display system info |
| `matrix` | Enter the matrix |
| `clear` | Clear terminal |
| `exit` | Close terminal |

**Project shortcuts:** `web-audio-player` · `earthquake-visualization` · `door-dashboard` · `veto-system`

## Projects

- **Web Audio Player** — Real-time audio visualizer with frequency spectrum and LUFS metering
- **Earthquake Visualization** — Interactive seismic data map for Puerto Rico
- **DoorDash Analytics** — Delivery metrics dashboard
- **Veto Voting System** — Team decision-making platform
